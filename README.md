# Data-Analytics-task1-EDA-Analysis-
# 🛒 Retail Sales Data Analysis using Python

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a Retail Sales dataset using Python. The objective is to analyze sales performance, customer behavior, product trends, and yearly/monthly sales patterns through statistical analysis and data visualization.

The project includes data cleaning, descriptive statistics, correlation analysis, time-series analysis, customer segmentation, product category analysis, and an interactive dashboard created using Matplotlib.

---

## 📂 Dataset

The dataset contains retail transaction information including:

- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount

---

## 🎯 Project Objectives

- Clean and preprocess the retail sales dataset.
- Perform descriptive statistical analysis.
- Analyze monthly and yearly sales trends.
- Identify top customers based on sales.
- Analyze sales by gender and product category.
- Study age distribution of customers.
- Visualize sales trends using different charts.
- Build a dashboard summarizing important business insights.
